# kswitch

### Get SWHKD (optional):
- `git clone https://github.com/waycrate/swhkd;cd swhkd`
- `make setup`
- `make clean`
- `make`
- `sudo make install`
- `sudo chown root:root swhkd`
- `sudo chmod u+s swhkd`
- `swhks &` <-- bind these to autorun
- `swhkd` <-- bind these to autorun
- `vim ~/.config/swhkd/swhkdrc` <-- edit your shortcuts
